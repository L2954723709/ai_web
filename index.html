光核闪避图鉴

操控光核活到更高波次，拾取道具强化输出，击破每波 Boss 后进入短暂休整。数值会随波次、难度和存活时间提升。

01基础玩法
🎮移动WASD / 方向键移动；电脑端鼠标悬停可跟随操控；手机全屏左手摇杆。
⚡冲刺空格 / 量子冲刺按钮。冷却 2.6 秒，命中范围内敌人造成小爆发，短暂无敌。
◎脉冲Q / 释放脉冲按钮。能量 100% 后可用，范围约 220，普通怪高伤，Boss 受到 190 + 波次 × 18。
⏸暂停P / ESC 暂停。击破 Boss 后休整约 3.9 秒，下一波刷新并补充强力道具。
02答题复活与防沉迷
⌁复活挑战死亡后可进入随机 Python 选择题挑战，积分达到 5 分后原波次复活，并获得 3 秒保护。
20计分规则每题 20 秒；答对 +1，答错或超时 -1，最低保持 0；提交答案后查看解析，点击“下一题”继续；达到 5 分后点击“完成挑战”。
⏱学习换时活跃游玩达到 60 分钟后暂停游戏，通过同样的 5 分挑战可增加 15 分钟。
↻进度保存答题积分、当前题目、倒计时、死亡状态和游玩时长保存在本机，刷新不能清零。
03成长与刷新

独立进化：每种道具首次拾取为 R1，再次拾取进化到 R2、R3；满阶后继续拾取会触发满阶共振，补充脉冲和分数。

品质：标准为基础效果；强化额外提高约 20% 强度与 15% 时长；超频再提高约 20%。强化第 3 波后稳定开放，超频第 7 波后开放。

公平叠加：持续效果重复拾取只追加约 58% 新时长，并受独立上限约束，无法无限堆叠；R2 / R3 主要强化形态、数量和联动。

后期补偿：稀有品质有保底累积；波次越高，神器和高品质道具越常见，抵消敌人血量、攻击与数量成长。

小兵成长：血量约为基础值 × 1.08 + 0.21 × (波次 - 1) + 存活时间加成，并根据已获得的输出道具同步强化；第 3 波起可能出现带护甲的冠军单位。

Boss 成长：基础 HP 叠加波次、时间、难度和玩家火力倍率；普通难度约 0.96 倍，地狱约 1.36 倍。Boss 拥有韧性护盾、抗爆发上限，低血量时技能频率提升。

神器与品质开放 第 3 波起：强化品质进入主要掉落池 第 5 波起：新星 / 黑洞 / 陨星 / 狂热 第 7 波起：超频品质、神盾 / 时停 / 开奖 第 9 波起：超载 / 奇迹，神器概率随波次提升
七组共鸣 刀刃 + 疾行：风切推进 弹幕 + 分裂弹：镜像弹仓 护盾 + 反伤甲：反应装甲 冻结 + 光矛：极寒光矛 牵引 + 黑洞：奇点引擎 生命 + 虹吸：再生血契 充能 + 涌能：零延迟回路
